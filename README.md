MERN Stack Course Project

## Team

| Padhu  | Module 1         | Auth (login/register), JWT middleware, Admin dashboard |

- Sushma: Navbar + React Router config

## Phase 2 (parallel, after Padhu shares middleware)
- Tej: Full booking system
- Sushma: FCFS resources + Lab timetable

## Setup
1. Clone the repo
2. Copy .env.example to .env in both /server and /client and fill in values (ask Tej for the MongoDB URI)
3. cd server then npm install
4. cd client then npm install
5. Run server: cd server then node index.js
6. Run client: cd client then npm run dev
